# 05 Task Spec：单次施工任务规格

Task Spec 是研发中期最高频使用的文件。它的目标是避免每次把完整 Product Brief 和 AI-SDD 都塞给 Codex。

## 使用时机

- 已经进入研发期
- 每一次要让 Codex 改代码前
- Bug 修复、普通功能、小 UI 修改、部署调整前
